# 背景图库目录

建议把网站背景图按主题放在这里：

- `campus/`：校园、教室、图书馆、操场等
- `nature/`：山、湖、云、森林等自然背景
- `geography/`：地理、地图、地球、卫星等
- `sports/`：跑道、球场、体育场等
- `textures/`：噪声、纸张、抽象纹理等

推荐规格：

- 横向图片优先
- 建议尺寸不低于 `1920x1080`
- 推荐压缩为 `.webp`，也可以先使用 `.jpg`
- 避免明显人脸、商标、真实学校 Logo 和未经授权素材
- 命名建议：`campus-sunlight-01.webp`、`library-study-01.webp`

如果要登记图片信息，请同步维护：

`src/data/backgrounds.json`
